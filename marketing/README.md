# Haqqi Marketing Package

This folder contains launch-ready marketing assets for `حقي | Haqqi`.

## Structure

- `assets/` shared brand files and logo
- `banners/` HTML social and ad creatives sized for capture/export
- `brochure/` printable HTML brochure designed for PDF export
- `presentation/` HTML pitch deck slides
- `video/` scripts and storyboard markdown
- `social/` captions, hashtags, and 30-day content calendar
- `email/` bilingual HTML email templates

## Usage

1. Open any HTML file in a browser.
2. Capture banners at 100% zoom for the target dimensions.
3. Print `brochure/haqqi-brochure.html` to PDF with margins disabled or minimal.
4. Present `presentation/haqqi-pitch-deck.html` in browser full-screen or export slides manually.
